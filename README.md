# UAS Praktikum Web Dasar

Proyek ini merupakan tugas Ujian Akhir Semester (UAS) untuk mata kuliah Praktikum Web Dasar di Universitas PGRI Madiun (UNIPMA). Proyek ini mencakup implementasi dasar pengembangan web menggunakan HTML, CSS, PHP (MySQL), dan pengelolaan database.

## Fitur Utama

Proyek ini mencakup folder praktikum berikut:
- **Layouting & Desain**: Implementasi HTML5 dan CSS3 untuk struktur halaman web.
- **Sistem CRUD**: Manajemen data mahasiswa (Tambah, Tampil, Ubah, Hapus) yang terhubung ke database.
- **Pemrosesan Form**: Penanganan input data melalui form PHP.
- **Logika PHP**: Penggunaan string, tanggal, dan fungsi dasar PHP lainnya.

## Teknologi yang Digunakan

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: PHP (Legacy MySQL Extension)
- **Database**: MySQL (Database Name: `unipma`)
- **Server**: Laragon / XAMPP
